I built this website to document my Google Summer of Code 2024 work with DBpedia week by week.

The journal records technical progress, design decisions, mentoring discussions, setbacks, and lessons from developing ontology versioning and package-management infrastructure in an open-source community. It makes the process behind the final software visible, not only the result.
